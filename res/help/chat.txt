 **ENTER**              Send message
 **SHIFT+ENTER**        Add newline to message text

 **CTRL+F**             Upload/Send file
 **CTRL+D**             Download file(s)

 **CTRL+X**             Close conversation
 **CTRL+H**             Show this helptext

 **PAGE+UP**            Scroll messages up
 **PAGE+DOWN**          Scroll messages down


 **/delete-messges**    Delete all stored messages for this
                    conversation.
 **/search <pattern>**  Search for a pattern in all stored
                    messages.
 **/export <path>**     Export all messages to an unencrypted
                    text file.
